# User Manual - Toko Roti Jiya (Jeycookie)

## 1. Panduan untuk Pelanggan

### 1.1 Melihat Produk
1. Buka website Jeycookie
2. Di halaman utama, Anda akan melihat produk-produk unggulan
3. Klik **"Products"** di menu untuk melihat semua produk
4. Gunakan filter kategori untuk memilih jenis produk (Cookies, Donuts, Cakes, dll)
5. Gunakan kotak pencarian untuk mencari produk tertentu

### 1.2 Menambah ke Keranjang
1. Pilih produk yang ingin dibeli
2. Klik tombol **ðŸ›’ (ikon keranjang)** pada produk
3. Produk akan otomatis masuk ke keranjang
4. Angka di badge keranjang akan bertambah

### 1.3 Melihat Keranjang
1. Klik **"Cart"** di menu navigasi
2. Anda akan melihat daftar produk yang sudah dipilih
3. Ubah jumlah (quantity) jika diperlukan
4. Klik **"Remove"** untuk menghapus item
5. Total harga akan terupdate otomatis

### 1.4 Checkout (Tanpa Login / Guest)
1. Dari halaman keranjang, klik **"Checkout"**
2. Isi formulir:
   - Nama Lengkap
   - Email
   - No. HP
   - Alamat Pengiriman
3. Klik **"Lanjut ke Pembayaran"**
4. Pilih metode pembayaran (GoPay, QRIS, Bank Transfer, dll)
5. Ikuti instruksi pembayaran
6. Setelah berhasil, Anda akan menerima email konfirmasi

### 1.5 Checkout (Dengan Login)
1. Klik **"Login"** dan masuk ke akun Anda
2. Lakukan checkout seperti biasa
3. Data alamat akan terisi otomatis dari profil
4. Riwayat pesanan akan tersimpan di akun Anda

### 1.6 Melihat Riwayat Pesanan
1. Login ke akun Anda
2. Klik nama Anda di pojok kanan atas
3. Pilih **"My Orders"**
4. Lihat status pesanan Anda:
   - **Pending**: Menunggu pembayaran
   - **Paid**: Sudah dibayar
   - **Processing**: Sedang diproses
   - **Shipped**: Sedang dikirim
   - **Completed**: Selesai

### 1.7 Registrasi Akun
1. Klik **"Login"** di menu
2. Klik **"Daftar Sekarang"**
3. Isi formulir:
   - Nama Lengkap
   - Email
   - No. HP (opsional)
   - Password (min 8 karakter)
   - Konfirmasi Password
4. Klik **"Register"**

## 2. Panduan untuk Admin

### 2.1 Login Admin
1. Akses halaman login
2. Masukkan email dan password admin
3. Centang **"Login sebagai Admin"** (jika ada)
4. Klik **"Login"**
5. Anda akan diarahkan ke Dashboard Admin

### 2.2 Dashboard
Dashboard menampilkan:
- Total Produk
- Total Pesanan
- Total Pelanggan
- Total Revenue
- Pesanan terbaru
- Produk dengan stok rendah
- Grafik revenue 7 hari terakhir

### 2.3 Kelola Produk

#### Menambah Produk Baru:
1. Klik **"Products"** di sidebar
2. Klik tombol **"+ Add Product"**
3. Isi formulir:
   - Nama Produk
   - Kategori
   - Deskripsi
   - Harga
   - Stok
   - Berat (opsional)
   - Upload Gambar
   - Aktif (toggle)
   - Featured (toggle)
4. Klik **"Save"**

#### Mengedit Produk:
1. Di halaman Products, klik **"Edit"** pada produk
2. Ubah data yang diperlukan
3. Klik **"Update"**

#### Menghapus Produk:
1. Klik **"Delete"** pada produk
2. Konfirmasi penghapusan

### 2.4 Kelola Kategori
1. Klik **"Categories"** di sidebar
2. Tambah/Edit/Hapus kategori sama seperti produk

### 2.5 Kelola Pesanan

#### Melihat Pesanan:
1. Klik **"Orders"** di sidebar
2. Lihat daftar semua pesanan
3. Filter berdasarkan status jika diperlukan

1. Klik pada pesanan untuk melihat detail

1. Klik nama admin di pojok kanan atas

4. Klik **"Confirm Logout"**

## 3. FAQ

**Q: Bagaimana cara melacak pesanan saya?**
A: Login ke akun Anda, lalu klik "My Orders" untuk melihat status pesanan.

**Q: Metode pembayaran apa saja yang tersedia?**
A: GoPay, QRIS, Bank Transfer (BCA, Mandiri, BNI, BRI), Kartu Kredit/Debit.

**Q: Berapa lama pesanan diproses?**
A: Pesanan akan diproses dalam 1-2 hari kerja setelah pembayaran dikonfirmasi.

**Q: Bagaimana jika produk habis?**
A: Produk yang habis akan ditandai "Sold Out" dan tidak bisa ditambahkan ke keranjang.

**Q: Apakah bisa checkout tanpa membuat akun?**
A: Ya, Anda bisa menggunakan Guest Checkout tanpa perlu mendaftar.
